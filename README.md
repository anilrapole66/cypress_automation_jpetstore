# Cypress E-Commerce Automation Framework

This project demonstrates end-to-end automation testing using Cypress.

## Features
- Page Object Model design
- Custom Cypress commands
- Data-driven testing using fixtures
- Mochawesome HTML reports
- Screenshot and video capture

## Test Scenarios
1. User login
2. Product search
3. Add to cart
4. Checkout flow

## Tools Used
Cypress
JavaScript
Node.js
Mochawesome
